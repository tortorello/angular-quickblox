angular2-quickblox
=========

A minimal Node module providing Quickblox integration for Angular 2+.

## Installation

  npm install angular2-quickblox --save

## Usage

  import { QuickBlox } from 'angular2-quickblox';
  
  console.log(QB.version);

## Tests

  npm test
